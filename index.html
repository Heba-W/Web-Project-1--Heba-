        Different Computer Parts

        Central Processing Unit

        The Central Processing Unit, also known as the CPU, is like the brain of the computer. The CPU carries out instructions from software which allows the computer to run and preform programs. It's like a boss of a company, it directs other parts of the system by making decisions, and sending commands.


        Motherboard

        From the name, you can already tell, the Motherboard is like the "mother" of the computer. It's the main circuit that connects the other components together so they can communicate with eachother. The Motherboard is essential because it's the part that makes everything else work together.


        Random-Access Memory

        Random-Access Memory, also known as RAM, is a type of computer memory that takes information and data, and temporarily stores it. This helps the computer run smoothly. Also, computers with more RAM can do more tasks at once without slowing down the computer. It's kind of like your desk, (more space can allow you to do more work).
